Takes .bmp images and either darkens them, greyscales them, or brightens them.
x86, C++

use makefile to compile

Usage: ./imageCvt <-gr|-br|-dk> <inputFile.bmp> <outputFile.bmp>

-gr greyscales the image
-br brightens the image
-dk darkens the image

can use the provided images for inputFile.bmp

can use any filename as program will create the file if it doesn't exist. I use tmp.bmp which I also provided.
